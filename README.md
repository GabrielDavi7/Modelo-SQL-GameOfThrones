# Modelo SQL - Game of Thrones

Projeto acadêmico desenvolvido para a disciplina de **Banco de Dados** do **Instituto Federal do Norte de Minas Gerais (IFNMG)**.

O objetivo deste repositório é armazenar a implementação de um banco de dados relacional inspirado no universo de **Game of Thrones**, utilizando SQL para criação das tabelas, inserção de dados e consultas.

> 🚧 Projeto em desenvolvimento.
> O conteúdo deste repositório será atualizado conforme o andamento do projeto.

## Tecnologias

- SQL
- PostgreSQL (ou Oracle)
- Git
- GitHub
